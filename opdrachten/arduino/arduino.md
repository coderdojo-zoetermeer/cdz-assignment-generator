*Introductie*

In deze opdracht ga je werken met de ingebouwde LED van een Arduino UNO.
:::

________________________________________________________________________________

::: build

![Arduino UNO =400x](./assets/schema.svg){.float-right}
*Arduino aansluiten*

Sluit de arduino UNO aan met de kabel op je computer. Als dit de eerste keer is
vraag dan een mentor om je te helpen.

:::

*pagebreak*
________________________________________________________________________________

:::: program

*Open het Blink voorbeeld programma*

Start de Arduino IDE en open het voor voorbeeld Blink. Je kan deze vinden in het
menu **File -> Examples -> 01.Basics -> Blink**. Hieronder kan je zien hoe de
code eruit ziet.

Start het programma met de **Upload** button. De LED knippert nu.
